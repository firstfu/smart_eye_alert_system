# 智眼警示系統（Smart Eye Alert System）文件中心

## 產品簡介

智眼警示系統是一套專為養老院、醫療診所和私人住宅設計的智慧監控解決方案。本系統運用先進的電腦視覺技術和人工智慧，提供以下核心價值：

### 主要特點

- 即時跌倒偵測與警報通知
- 支援一般網路攝影機，無需特殊硬體
- 快速部署，操作簡單直覺
- 合理的價格定位

### 目標客戶

- 養老院（首要目標市場）
- 小型醫療診所
- 私人住宅業主

### 核心功能

- 智慧跌倒偵測
- LINE 即時通知
- 簡易管理介面
- 事件記錄與回放
